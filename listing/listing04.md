
Что выведет программа? Объяснить вывод программы.

```go
package main
func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()
	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет последовательность от 0 до 9 включительно и затем ошибку deadlock.
Чтобы избежать ошибки, необходимо после цикла закрыть канал.
```
